# Sistema de Quiz - SQL Server

Projeto de banco de dados desenvolvido em SQL Server para gerenciamento de quizzes acadêmicos.

## Tecnologias utilizadas

* SQL Server
* T-SQL
* Procedures
* Modelagem Relacional

## Funcionalidades

* Cadastro de cursos
* Cadastro de disciplinas
* Cadastro de questões
* Cadastro de alternativas
* Sistema de usuários
* Controle de pontuação
* Ranking de jogadores
* Questões aleatórias
* Procedures para CRUD completo

## Estrutura do projeto

```bash
database/
   quiz.sql
```

## Conceitos aplicados

* Primary Key
* Foreign Key
* Procedures
* SELECT
* INSERT
* UPDATE
* DELETE
* ORDER BY
* ROW_NUMBER
* CTE
* Relacionamentos entre tabelas
